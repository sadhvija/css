CSS INTERVIEW QUESTIONS

1.What is CSS? A:CSS is defined as Cascading Style Sheets.It is used for styling a webpage.
2.In how many ways we can create CSS styling? A:In 3 ways we can create CSS styling they are inline,internal and external.
3.What is the difference between inline,internal and external? A:Inline CSS:styles are written directly inside the tag, the property / value pairs are in quotes, styles apply only to the current tag
4.What is specificity among the 3 ways of styling? A:Inline CSS has highest specificity.Internal CSS is more specific than external CSS.
5.What is syntax of CSS? A:selector{ property:value; }
ex:body{background-color:red;}
6.What is Selector? A:Selector selects an element which we want to style.
7.How many types of selectors are there?what are they? A:There are 5 types of selectors they are:
1.Simple selectors.
2.Combinator selectors.
3.Pseudo-class selectors.
4.Pseudo-element selectors.
5.Attribute-selectors.
8.What are general selectors? A:class,id,*,grouping,type.
9.what is difference between pseudo class and pseudo element? A:class is used to style the element based on state(hover,checked,focus,active).
element is used to style the specific part of the element(first line,first letter,after,before).
10.What is ruleset? A: CSS rule set contains one or more selectors and one or more declarations. it gives instructions to web browser.
11.What is boxmodel? A:Box model is used to design. box modal is a container which wraps around every element.
12.What are boxmodel properties? A:Box model consists of 4 properties:content,padding,border,margin.
13.What is difference between padding and margin? A:Margin is used to create space around the element outside the border. Padding is used to create space around the element inside the border.
14.What is difference between outline and border? A:An outline is a line drawn outside the element's border. A border that goes around padding and content.
15.Types of colors we have in CSS? A:There are different types of color representation in CSS.
They are:1.Direct,2.rgb,3.rgba,4.Hexa,5.Hsl,6.Hsla.
16.What is opacity? A:Transparency of color or element .opacity value ranges between(0-1).
17.What are the shadows in css and how many are they? AShadows are used to add shadows to text and images in HTML document.They are box,text,drop shadow.
18.What are the gradients in CSS and how many are they? A:Gradients are combination of colours in background. They are:Linear gradient,Conic gradient,Radial gradient.
19.Difference between float and display? A:Display specifies how the element will element appears in organizational terms to the browser. Float tells more about the alignment of the element.
20.Difference between display:inline vs display:block vs display:inline-block? A:Inline: The display:inline converts block level elements to display inline elements. Block: The display:block converts inline elements to diplay as block level elements. Inline-Block: It converts Inline elements to block elements.
21.What is the difference between visibility:hidden and display:none? A:visibility:hidden the space exists in the document but it does not visible. display:none we dont have the space for the element.and also content does not exist in the document.
22.What is object fit and how many types we can fit the object? A:Object fit is used to fit the content in the container. Object values:cover,contain,fill,none.
23.What is background property? A:background property is used to maintain the styling of background. background-position,color,size,repeat,img.
24.What is the position property? A:Position property is used for positioning of element.
25.Types of position values we can give? A:5 types of position values:static,relative,fixed,absolute,sticky.
26.Difference between position:fixed and sticky? A:It depends, when you want that element simply doesnt move in your website then use fixed. But be careful with position: fixed, it can potentially cover other element. When you want that element scrolling into view and then stop at certain point, use sticky.
27.Difference between position:relative and absolute? A:Relative - the element is positioned relative to its normal position. Absolute - the element is positioned absolutely to its first positioned parent.
28.What is overflow in CSS? A:Overflow is used to control the overflown content on x and y axis.
29.What is z-index? A:z-index is used to maintain the stack order of element in vertical direction.(default value:0)
30.What is clear property? A:The clear property specifies what should happen with the element that is next to a floating element.
31.What are the CSS variables? A:CSS variables are used to maintain global CSS. CSS variables are used multiple times.
32.What is :root in css? A:The :root selector matches the document's root element.
33.What are text properties? A:Text color,Text-alignment,text decoration,text-transformation,text-spacing,text-shadow.
34.What are font properties? A:font-style,font-varient,font-weight,font-size,font-family.
35.Convert underline to linethrough? A:By giving offset value negative.
36.What is flexbox? A:Flexbox is a container which maintains the alignment of element in horizontal or vertical direction.
37.values of flex wrap? A:wrap,no-wrap,wrap-reverse.
38.What is grid layout system? A:grid layoutis a multi dimensional layout system. it aligns the content horizontally and vertically.
39.Difference between the flex and grid? A:Flex is a single dimension layout. grid is a multi dimensional layout. default direction of flex is row. default direction of grid is column.
40.What are media queries in CSS? A:media queries are used to define different style rules for different media types.
41.What are transitions in CSS? A:Transitions allow you to change the property value smoothly over a specified duration.
42.What are transforms in CSS? A:Transform applies a 2D or 3D transformation to an element.This property allow you to rotate,scale,skew,move.
43.what is difference between transitions vs transforms vs animations? A:Transition provide a change from one state to another. Animation sets multiple points of transition upon different keyframes. Transform applies a 2d or 3d transformation to an element.
44.What is matrix in transforms? A:The matrix() function is an inbuilt function which is used to create a homogeneous 2D transformation matrix.
45.Difference between CSS and CSS3? A: CSS is a fundamental technology used to style and format web pages. Supports only single text blocks. Does not support media queries. Not supported by all types of modern browsers. Requires manual development of rounded gradients and corners. CSS3 stands for Cascading Style Sheets Level 3. Supports multi-column text blocks. Supports media queries. Supported by all modern browsers. Provides advanced codes for setting rounded gradients and corners.
